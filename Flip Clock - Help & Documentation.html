‚ùì Troubleshooting

Clock not updating?

Refresh the page (F5 or Ctrl+R). Ensure JavaScript is enabled in your browser.

Fullscreen not working?

Some browsers require specific permissions. Check your browser settings or try the F key instead of the button.

Animations stuttering?

Close other tabs or applications to free up system resources. Hardware acceleration in browser settings may help.

Font size not changing?

Make sure you're not at the minimum (60%) or maximum (140%) limit. Try using the slider for fine control.

Controls not visible?

Press S to show the font control panel, or check if you're in fullscreen mode where controls are positioned differently.
